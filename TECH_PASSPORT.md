# Технический паспорт проекта: CalcHub (Финансовый Хаб)

Этот документ содержит исчерпывающее описание архитектуры, структуры данных и логики автоматизации репозитория. Предназначен для быстрого погружения в проект.

## 1. 🏗️ ОБЩАЯ АРХИТЕКТУРА И СТЕК ТЕХНОЛОГИЙ

*   **Концепция:** Serverless Static Web App (SSWA). Проект представляет собой статичный хаб калькуляторов. Расчеты выполняются на клиенте, динамические данные поставляются через JSON-файлы, версионируемые в Git.
*   **Стек технологий:**
    *   **HTML5 / ES Modules:** Чистый JavaScript без тяжелых фреймворков.
    *   **Tailwind CSS (CDN):** Стилизация в стиле «Strict Neo-Banking».
    *   **Chart.js (CDN):** Визуализация графиков и диаграмм.
    *   **html2canvas (CDN):** Генерация скриншотов для шеринга.
    *   **Service Worker (sw.js):** Network-first стратегия для оффлайн-работы и кеширования внешних библиотек.
*   **Хранение данных:** Динамические показатели (ставки, инфляция) хранятся в `rates.json`, `inflation.json` и `regions.json`.

## 2. 🗂️ КАРТА РЕПОЗИТОРИЯ И СВЯЗИ

### Структура файлов:
*   `index.html`: Главная страница-хаб с адаптивной сеткой карточек.
*   `*.html` (в корне): Отдельные страницы калькуляторов (Mortgage, Wealth, Millionaire и т.д.).
*   `js/`:
    *   `api.js`: Модуль загрузки внешних JSON данных.
    *   `theme.js`: Управление светлой/темной темой (localStorage).
    *   `utils.js`: Утилиты форматирования (Intl.NumberFormat) и маски ввода.
    *   `ui-utils.js`: Общая UI-логика (Toast, Screenshots, Metrika).
*   `dev_tools/`: Скрипты автоматической верификации UI (Playwright).
*   `update_*.js`: Скрипты автоматизации для обновления данных.
*   `rates.json`, `inflation.json`, `regions.json`: Базы данных проекта.

### Взаимосвязи:
`GitHub Action` -> `update_rates.js` -> `CBR API` -> `rates.json` -> `fetch() в калькуляторе` -> `UI / Chart.js`.

## 3. 🤖 РОБОТЫ И АВТОМАТИЗАЦИЯ (GITHUB ACTIONS)

*   **Workflow:** `.github/workflows/rates.yml` запускается ежедневно в 03:00 UTC.
*   **CBR API Transition:** Скрипты `update_rates.js` и `update_inflation.js` переведены с ИИ-поиска на **прямые HTTP GET запросы** к официальным сервисам ЦБ РФ:
    *   **DailyInfo ASMX:** Получение ключевой ставки в реальном времени.
    *   **XML Main Info:** Получение текущих макропоказателей инфляции.
*   **Причина перехода:** Использование Gemini Search приводило к частым ошибкам 429 (Rate Limit) и блокировкам IP на Free Tier. Прямое обращение к API обеспечило 100% стабильность.

## 4. 🔍 СКРЫТЫЕ И НЕАКТИВНЫЕ ЭЛЕМЕНТЫ

*   **Legacy AI:** Скрипт `update_regions.js` всё еще использует Gemini AI (google_search) для сбора статистики по зарплатам. Это временное решение, требующее перехода на статические источники.
*   **Удаленный код:**
    *   Полностью удалена интеграция Firebase (аналитика и БД) для упрощения и приватности.
    *   В `mortgage.html` удалены кнопки ручного обновления ставок (теперь только авто-обновление роботом).
*   **Fallback:** В скриптах обновления предусмотрены жестко заданные массивы исторических данных (инфляция с 2000 года) на случай недоступности серверов ЦБ.

## 5. 🎯 ТЕКУЩИЙ СТАТУС СТРАНИЦ И ФИЧ

### Полностью функциональные страницы:
*   **ИпотекаПро**: Расчет с досрочными платежами и вычетами.
*   **Рентген Доходов**: Сравнение с региональными бенчмарками.
*   **Аренда vs Ипотека**: Долгосрочное сравнение стратегий.
*   **Шредер Рубля**: Калькулятор накопленной инфляции.
*   **Машина или Такси**: Сравнение стоимости владения.
*   **Генетическое богатство**: Прогноз бюджета на ребенка.

### Последние ключевые фичи:
*   **Миллионер (`millionaire.html`)**: Расчет даты достижения цели с учетом ежемесячной капитализации. Вычисляет день недели и дает персонализированные советы по ускорению.
*   **Честная Ставка (`honest-credit.html`)**: Разоблачение скрытых страховок.
    *   **Logic:** Использование метода бисекции для расчета реальной эффективной ставки (EIR/IRR).
    *   **UX:** Динамические вердикты в зависимости от размера скрытой переплаты.
